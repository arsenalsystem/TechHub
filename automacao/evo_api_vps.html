Evolution API: Comece do Zero

Estratégia ideal para gastar o mínimo e ter performance profissional 24h por dia.

Custo estimado: R$ 25 a R$ 45 por mês.

1 Onde contratar a VPS mais barata e boa?

Fuja de empresas nacionais que cobram caro por pouca máquina. Vá direto para as duas gigantes alemãs que dominam o mercado de baixo custo:

Hetzner (Plano CX23)

Cerca de € 3,99/mês (~R$ 25,00)

Entrega 2 vCPUs e 4 GB de RAM. A opção mais rápida e barata do mercado.

Contabo (Cloud VPS 10)

Cerca de € 4,50/mês (~R$ 28,00)

Incríveis 8 GB de RAM e 4 vCPUs. Uma máquina gigante pelo preço de um lanche.

Nota: Para contratar, você só precisa de um cartão de crédito internacional (Nubank, Inter, etc.). Escolha o sistema operacional Ubuntu 24.04 LTS na hora de criar.

2 O "Pulo do Gato" para instalar

Em vez de comandos complexos de terminal, use ferramentas gratuitas como o Coolify ou o CasaOS.

O Coolify é uma alternativa open-source incrível. Você instala ele na sua VPS vazia com apenas um comando e ele se transforma em um painel visual no seu navegador.

Como funciona: Dentro do Coolify, a Evolution API já vem pronta no catálogo. Basta clicar em "Instalar" e o painel configura automaticamente:
  • Banco de dados (PostgreSQL/Redis)
  • Download da API
  • Geração de endereço seguro HTTPS com cadeado

3 O que você precisa além da VPS?

Para a API funcionar, o WhatsApp exige uma conexão segura (https://). Você precisará de um Domínio.

Onde comprar:

  • Registro.br: Cerca de R$ 40,00/ano para extensões .com.br.
  • Namecheap/Hostinger: Extensões como .click, .site ou .online custam menos de R$ 10,00 no primeiro ano.

Apontamento:

Crie uma conta gratuita na Cloudflare para gerenciar o seu domínio. É ela quem vai conectar o nome do seu site ao IP da sua VPS de forma segura.
