Estratégia de Contexto Persistente

Use este guia para manter a continuidade entre sessões de chat temporário, economizando tokens e tempo.

1. O Checkpoint de Fechamento

Antes de fechar a sessão, peça para a IA gerar o "DNA" da conversa:

"Gere um resumo técnico conciso (Markdown) para carregar em uma nova sessão. Inclua: Objetivos, Decisões, Variáveis e Próximos Passos."

2. Formatos de Arquivo Sugeridos

Salve os dados em um arquivo .txt ou .md usando estruturas que a IA lê rapidamente:

  • Markdown: Para hierarquia e tópicos.
  • JSON: Para dados brutos e parâmetros técnicos.
  • Prompt Comprimido: Para máxima economia de espaço.

3. O Bootstrap (Reinicialização)

Ao iniciar o novo chat, use este comando de abertura:

"Aja como meu assistente. Assuma o contexto abaixo e ignore saudações:

[COLE O CONTEÚDO DO SEU ARQUIVO AQUI]"

4. Dicas de Ouro

✦ Use a "Compressão de Prompt" para reduzir textos longos em instruções densas que apenas IAs interpretam bem.

✦ Mantenha um arquivo fixo de "Persona" com suas preferências de escrita e regras que nunca mudam.
